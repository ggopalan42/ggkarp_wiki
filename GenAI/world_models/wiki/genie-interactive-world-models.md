# Generative / Interactive World Models (the current wave)

Models that learn controllable, playable environments, increasingly from video.

- **GameGAN** (2020) and especially **Genie** — "Generative Interactive Environments" (DeepMind, 2024) and Genie 2/3. Learn controllable, playable environments from video without action labels.
- **DIAMOND** — "Diffusion for World Modeling" (2024). Uses diffusion as the dynamics model; improves visual fidelity and downstream RL.
- **Decision Transformer** (Chen et al., 2021) and **Trajectory Transformer** — reframe control as sequence modeling; adjacent but important context.

- **Framework placement:** see [[four-axis-reading-framework]] — generative observation representation, transformer/diffusion dynamics, used as a playable environment, largely self-supervised from video.
- **Stay current:** DeepMind blog (Genie), NVIDIA blog (Cosmos).
